# Signal Object Detection — Kaggle Competition

This repository contains the solution for the **Signal Object Detection** challenge on Kaggle, where the goal is to detect and count objects within noisy radio signals.

## 🏆 Result
* **Rank:** #49
* **Accuracy:** 77.115%

## 🔗 Competition Link
You can find the official competition page here: 
[Kaggle — Signal Object Detection](https://www.kaggle.com/competitions/signal-object-detection)

## 📌 Project Overview
The project focuses on parsing noisy RF (Radio Frequency) signals to determine the exact number of objects present. The core technical challenge involved structuring the model pipeline and experimenting with classification vs. regression heads to optimize the final counting accuracy.
